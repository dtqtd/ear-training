# 听见 · 视唱练耳

一个纯前端的视唱练耳应用：包含旋律音准检测和和弦级数判断。旋律模式会随机播放音符序列，通过麦克风检测演唱音高，并按音分偏差逐音评分；和弦模式会随机播放和弦进行，让你判断每个和弦的级数。

## 运行

在当前目录执行：

```bash
python3 -m http.server 8000
```

然后打开 `http://localhost:8000`。

麦克风权限通常只允许在 `localhost` 或 HTTPS 页面使用，因此不建议直接双击 `index.html`。

## 使用方法

1. 选择训练模式。
2. 旋律音准：选择音符数量、音域、难度和评分标准；音符数量和难度都支持“任意”。
3. 和弦级数：选择和弦数量、和弦难度和调性。三和弦、七和弦按大调自然和声出题；复杂模式会混合大小、属、减、挂留和扩展和弦。
4. 点击“播放题目”。

## 旋律音准

1. 点击“播放题目”，记住听到的音。
2. 戴上耳机，点击“开始演唱”并允许麦克风权限。
3. 按照听到的乐句，连续、自然地哼出全部音符，不需要在单个音上等待。
4. 唱完后点击“结束并评分”，查看整句音准准确率和逐音偏差。

检测过程中可以随时点击“结束并评分”。输入音量条用于排查麦克风：

- 音量条不动：浏览器没有收到麦克风信号，请检查系统输入设备和输入音量。
- 音量条移动但不显示音名：声音已收到，请靠近麦克风并持续唱一个稳定的“啊”。

首次允许麦克风后，“换一题”和“下一题”会复用同一个麦克风连接，不会重复申请权限。真正刷新或关闭网页时，应用会释放麦克风和音频上下文；手机浏览器是否在下次打开时再次询问，取决于该浏览器的站点权限设置。

## 和弦级数

1. 点击“播放题目”。应用会先播放主音和 I 和弦建立调性感，再播放和弦进行。
2. 为每个和弦选择 I、ii、iii、IV、V、vi、vii°。三和弦模式中 I、IV、V 是大三和弦，ii、iii、vi 是小三和弦，vii° 是减三和弦；七和弦模式对应 Imaj7、ii7、V7、viiø7 等自然大调七和弦。
3. 点击“提交答案”，查看准确率和逐项答案。

音频只在浏览器本地处理，不会录制或上传。
